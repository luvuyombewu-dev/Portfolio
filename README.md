# 💻 Luvuyo Mbewu - Developer Portfolio

A modern, responsive personal portfolio website built with **React and Vite** to showcase my technical skills, projects, certifications, experience, and professional journey as a **Computer Engineering graduate and software developer**.

The portfolio focuses on clean UI design, responsive layouts, smooth animations, and modern frontend development practices.

# ⚙️ Installation & Setup

## Clone Repository

```bash
git clone https://github.com/luvuyombewu-dev/Portfolio.git
```

## Enter Project Folder

```bash
cd Portfolio
```

## Install Dependencies

```bash
npm install
```

## Start Development Server

```bash
npm run dev
```

Application runs at:

```text
http://localhost:5173
```

---

# 🏗️ Build For Production

Create production build:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
